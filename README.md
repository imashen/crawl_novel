# crawl_novel
晋江文学城(www.jjwxc.net)爬虫程序，输入novelid即可爬取，自动保存

## 安装依赖
`pip install bs4`
</br>
`pip install requests`

## 使用程序
`python do.py`
输入想要爬取的小说novelid即可自动爬取

## 示例
链接：https://www.jjwxc.net/onebook.php?novelid=452296 </br>
其中novelid即为452296 </br>
